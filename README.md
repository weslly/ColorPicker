## Installation
Install this repository via [Package Control](http://wbond.net/sublime_packages/package_control)


## Usage
To insert or change a selected color, use:

- Linux: `ctrl+shift+c`
- Windows: `ctrl+shift+c`
- OS X: `cmd+shift+c`
